--------------------------------
## RSS
访问/r目录即可.  

--------------------------------
## 关于安装  
1. 根目录下解压后界面就可以访问 
2. 访问/a/，会显示让你用指定方式生成通行证，之后登陆即可发布文章/页面.
3. 修改/a/conf.php文件以达到配置的目的。
然后 ...好好享受♂吧！ 

--------------------------------
